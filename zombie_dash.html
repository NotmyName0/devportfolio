Zombie Dash

Built with GameMaker Studio

Download Project

Project Summary

Zombie Dash is a game where the player fights increasingly diffcult waves of zombies. It features health bars, a gun with animated bullets, and sprite animations. Inspired by Bloon Tower Defense, the game was created during a game jam and later improved over five weeks. Challenges included creating health bars and implementing a wave-based zombie spawn system. The game is available on Windows 10, with compatibility for other operating systems untested.

Challenges and solutions

Health bars

Creating health bars was a challenging aspect of the game development process. One major challenge was figuring out how to assign a health bar to both the player and each individual zombie in the game. Through this challenge, I was able to improve my understanding of objects in GameMaker Studio 2 and develop more advanced code for creating health bars and other related elements.

Assigning hit points and setting the width

Initially, I faced the issue of how to assign hit points without increasing the width of the health bar. To address this, I learned to divide the hit points by the maximum hit points to create a percentage, then multiply the percentage by the maximum health bar width to get the actual width of the health bar.

Spawning zombie instances

The challenge was to figure out how to spawn zombies equally from multiple spawn points in the game. Initially, I tried to divide the number of spawn points by the number of zombies to be spawned in each wave. However, this proved to be difficult to implement, especially when I had an unequal number of spawn points or zombies to spawn. This made it more difficult to code. So, I decided to simplify the code by randomly choosing a spawn point to spawn from.
