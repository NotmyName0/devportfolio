Email Archiver

Built with Python

Project Summary

Driven by a desire to continuously back up and protect my data, I decided to create my own Python script to download my emails. This allows me to always have a local copy in case anything goes wrong.

Currently the project is still in development.

Features and functionalities

    Backup Types

  • Full backups and increment backups

    • Database

  • The program used an SQL Lite database to store sensitive information such as client id and client secret

Technologies used

SQLAlchemy

SQLAlchemy is an Object Relational Mapper for the Python language; it is used to map Python classes to relational database tables.

Microsoft Azure

Microsoft Azure provides the necessary OAuth 2.0 authorization, allowing the script to access the user's Outlook account.

OAuth 2.0

OAuth 2.0 authorization framework is a protocol that allows a user to grant third-party websites or applications access to their personal information, such as their username or date of birth

Visual Studio Code

Visual Studio Code is an integrated development environment that is highly customizable. It offers ease of use and quick setup.

Python

Python is a simple programming language. I have chosen this language for my project as it is easy to work with, avoiding unnecessary complexities.

SQLiteStudio

SQLiteStudio is a relational database management system, similar to Microsoft SQL Server. It features a simple and responsive interface, which aligns well with my approach to software development.

Version control Github

A widely used version control system that enables developers to track changes to their code over time, collaborate with others on the same project, and roll back to previous versions if necessary.

Challenges and solutions

Designing a database

When I first started designing the SQLite database, I experienced difficulty in defining the relationships between each entity. Using what I learned from TAFE, I decided to explore Object-Relational Mapping (ORM) tools, and one of them was SQLAlchemy. With SQLAlchemy, I was able to avoid focusing too much on the details and instead concentrate on the actual data itself.

Working With 7zip Files

When I first started developing a function to create and write to a 7zip file, I experienced difficulty trying to create a function that worked with both text files and 7zip files. I later decided to split the code into two separate functions: one for handling text files and the other for handling 7zip files.
